### Intro

A mini-redis, supporting `get`, `set (with ttl)`, `del`, `publish`, `subscribe`.

### How to build

```bash
cargo update
cargo build
```

### How to run

```bash
cargo run --bin server
cargo run --bin client
```

### Usage

Type `help` in client will get the command list.
